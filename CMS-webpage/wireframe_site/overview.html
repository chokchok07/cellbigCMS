Overview

이 정적 사이트는 `CMS-webpage/wireframe` 폴더의 각 화면을 간단한 HTML/CSS로 구현한 예시입니다.

  • Dashboard — 대시보드 레이아웃
  • Content List — 콘텐츠 리스트
  • Content Editor — 콘텐츠 에디터 인터페이스
  • Version Register — 버전 등록 팝업
  • Product List — 제품 리스트
  • Product Detail — 제품 상세
  • Product Editor — 제품 등록/편집
  • Package List — 패키지 리스트
  • Package Editor — 패키지 등록/편집
  • Package Detail — 패키지 상세
  • Device List — 디바이스 리스트
  • Device Detail — 디바이스 상세 정보
