## 목적
개별 제품의 상세 정보, 연결된 패키지, 포함된 콘텐츠를 확인하고 관리할 수 있는 화면입니다.

## 레이아웃 구조

### 상단 바
- 제품 선택 드롭다운: 다른 제품으로 빠른 전환
- 전역 검색: 시스템 전체 검색
- 알림 아이콘: 배포/업데이트 알림 표시
- 사용자 메뉴: 프로필, 설정, 로그아웃

### 좌측 사이드바
- 대시보드, 콘텐츠, 패키지, 제품(활성화), 디바이스, 리포트, 설정 메뉴

### 메인 영역

#### 1) 헤더
- 페이지 제목: "🏢 Product Detail: {제품명}"
- [Edit] 버튼: 제품 정보 편집
- [Delete] 버튼: 제품 삭제 (확인 모달)
- 생성일 표시

#### 2) 기본 정보 섹션
- **Product ID**: 제품 고유 ID (읽기 전용)
- **Name**: 제품명
- **Description**: 제품 설명
- **Default Package**: 기본 제공 패키지 ID
- **Created**: 생성일시
- **Updated**: 최종 수정일시

#### 3) 연결된 패키지 섹션
- 제목: "📦 연결된 패키지 (개수)"
- [+ Add Package] 버튼: 패키지 추가
- 패키지 테이블:
  - Package Name: 패키지 이름
  - Version: 최신 버전
  - Deploy Date: 최초 배포일
  - Action: [View] (패키지 상세로 이동), [Remove] (연결 해제)

#### 4) 포함된 콘텐츠 섹션
- 제목: "📝 포함된 콘텐츠 (개수)"
- 이 제품의 패키지들에 포함된 모든 콘텐츠 목록 (중복 제거)
- 콘텐츠 테이블:
  - Content Name: 콘텐츠 이름
  - Type: single / container / child / mixed
  - Version: 최신 버전
  - Status: Published 🟢 / Staged 🟡 / Draft ⚪ / Deprecated ⚫

## 주요 상호작용 플로우

### 1. 제품 정보 편집
1. [Edit] 버튼 클릭
2. 편집 모드 전환 또는 편집 페이지로 이동
3. 제품명, 설명, 기본 패키지 수정
4. 저장 후 상세 페이지로 복귀 (변경사항 반영)

### 2. 제품 삭제
1. [Delete] 버튼 클릭
2. 확인 모달 표시:
   - "정말 삭제하시겠습니까?"
   - 연결된 패키지 수 표시
   - "이 작업은 되돌릴 수 없습니다"
3. 의존성 검사:
   - 연결된 패키지가 있으면 경고 및 삭제 차단
   - 또는 "Force Delete" 옵션 제공 (위험 경고)
4. 확인 → 삭제 수행 → 제품 리스트로 복귀

### 3. 패키지 추가
1. [+ Add Package] 버튼 클릭
2. 패키지 선택 모달 표시
3. 기존 패키지 목록에서 선택
4. 추가 → 연결된 패키지 섹션에 표시

### 4. 패키지 제거
1. 특정 패키지의 [Remove] 버튼 클릭
2. 확인 모달: "이 패키지를 제품에서 제거하시겠습니까?"
3. 확인 → 연결 해제 (패키지 자체는 삭제되지 않음)

### 5. 패키지 상세 보기
1. 특정 패키지의 [View] 버튼 클릭
2. 패키지 상세 페이지로 이동

### 6. 콘텐츠 보기
1. 특정 콘텐츠의 Content Name 클릭
2. 콘텐츠 상세 페이지로 이동

## 컴포넌트 및 검증

### 기본 정보 섹션
- Product ID는 읽기 전용 (불변)
- Name과 Description은 편집 가능
- Default Package는 연결된 패키지 중에서 선택

### 연결된 패키지 섹션
- 패키지 추가 시 중복 방지 (이미 연결된 패키지는 선택 불가)
- 패키지 제거 시 기본 패키지(Default Package)인 경우 경고
- 최소 1개 이상의 패키지 유지 권장

### 포함된 콘텐츠 섹션
- 이 제품의 모든 패키지에 포함된 콘텐츠를 집계
- 중복 제거 (같은 콘텐츠가 여러 패키지에 포함된 경우 1번만 표시)
- 콘텐츠 버전은 최신 버전 표시

## 에러 및 엣지 케이스 처리

### 제품 없음
- 잘못된 Product ID로 접근 시:
  ```
  ⚠️ Product not found
  The product you're looking for doesn't exist or has been deleted.
  [Back to Product List]
  ```

### 삭제 불가
- 연결된 패키지가 있는 제품 삭제 시도:
  ```
  ❌ Cannot delete product
  This product has connected packages.
  Please remove all packages before deleting.
  [View Packages] [Cancel]
  ```

### 패키지 제거 경고
- 기본 패키지 제거 시도:
  ```
  ⚠️ Warning
  This is the default package for this product.
  Removing it may affect device deployments.
  [Select New Default] [Cancel] [Remove Anyway]
  ```

## UX 노트
- 제품 상세는 제품과 관련된 모든 정보를 한 화면에서 확인할 수 있는 중심 허브입니다
- 연결된 패키지와 콘텐츠를 한눈에 파악할 수 있습니다
- 각 섹션에서 관련 엔티티의 상세 페이지로 빠르게 이동할 수 있습니다
- 편집/삭제 액션은 명확한 확인 절차를 거칩니다

## 반응형 고려사항
- 모바일: 섹션을 세로로 배치, 테이블은 카드 형태로 변환
- 태블릿: 2열 그리드 레이아웃, 섹션 토글 기능 추가
- 데스크톱: 전체 섹션 표시, 스크롤 가능

## 접근성
- 키보드 내비게이션 지원
- 스크린리더 지원: 각 섹션, 테이블, 버튼에 적절한 레이블
- 색상만으로 정보를 전달하지 않음 (아이콘 + 텍스트 병행)
