## 목적
관리자가 시스템에 등록된 모든 제품을 확인하고, 검색·필터링·편집·삭제 작업을 수행할 수 있는 제품 관리의 중심 화면입니다.

## 레이아웃 구조

### 상단 바
- 제품 선택 드롭다운: 특정 제품 선택 (전역 네비게이션)
- 전역 검색: 시스템 전체 검색
- 알림 아이콘: 배포/업데이트 알림 표시
- 사용자 메뉴: 프로필, 설정, 로그아웃

### 좌측 사이드바
- 대시보드, 콘텐츠, 패키지, 제품(활성화), 디바이스, 리포트, 설정 메뉴
- 현재 페이지: 제품 (🏢 Product) 활성화 상태

### 메인 영역

#### 1) 헤더
- 페이지 제목: "🏢 Product Library"
- [+ New Product] 버튼: 신규 제품 등록 화면으로 이동

#### 2) 필터 패널
- **검색**: 키워드로 제품명, 설명, Product ID 검색
- **[Apply Filter]** 버튼: 필터 조건 적용

#### 3) 일괄 작업 (Bulk Actions)
- **☑ Select All**: 현재 페이지의 모든 항목 선택
- **Bulk Actions 드롭다운**: 
  - Delete (일괄 삭제 - 확인 모달 필요)
- **[Apply]** 버튼: 선택된 작업 실행

#### 4) 제품 리스트 테이블
컬럼 구성:
- **☑ (체크박스)**: 개별 선택
- **Title**: 
  - 제품명 (대표 이름)
  - 제품 설명 (한줄 요약)
  - Product ID 표시

- **Packages**: 연결된 패키지 개수
- **Devices**: 설치된 디바이스 개수
- **Actions**:
  - [Edit]: 제품 정보 편집
  - [View]: 제품 상세 페이지로 이동
  - [Del]: 삭제 (확인 모달)

#### 5) 페이지네이션
- "Showing 1-3 of 8": 현재 표시 범위 및 전체 개수
- [◀ Previous] [1] [2] [3] [Next ▶]: 페이지 내비게이션

## 주요 상호작용 플로우

### 1. 제품 검색 및 필터링
1. 검색어 입력 또는 필터 조건 선택
2. [Apply Filter] 클릭
3. 리스트가 조건에 맞게 갱신됨
4. URL 쿼리 파라미터 업데이트 (북마크/공유 가능)

### 2. 신규 제품 등록
1. [+ New Product] 버튼 클릭
2. 제품 정보 입력 폼 표시 (모달 또는 전용 페이지)
3. 필수 정보 입력:
   - Product ID (자동 생성 또는 수동 입력)
   - 제품명 (name)
   - 설명 (description)
   - 기본 패키지 선택 (defaultPackageId, optional)
4. 저장 후 리스트로 복귀

### 3. 제품 정보 편집
1. 특정 제품의 [Edit] 버튼 클릭
2. 제품 정보 편집 폼 표시
3. 수정 후 저장
4. 리스트로 복귀 (변경사항 반영)

### 4. 제품 상세 보기
1. 특정 제품의 [View] 버튼 클릭 또는 제품명 클릭
2. 제품 상세 페이지([product-detail](./product-detail.md))로 이동
3. 연결된 패키지 목록, 설치된 디바이스 목록, 콘텐츠 목록 확인
4. 뒤로가기로 리스트 복귀

### 5. 일괄 작업 (Bulk Actions)
1. 여러 제품 체크박스 선택 (또는 Select All)
2. Bulk Actions 드롭다운에서 작업 선택 (Delete)
3. [Apply] 클릭
4. 확인 모달 표시 (특히 삭제 시)
5. 확인 후 작업 실행
6. 성공/실패 토스트 메시지 표시
7. 리스트 자동 갱신

### 6. 제품 삭제
1. 특정 제품의 [Del] 버튼 클릭
2. 확인 모달: "정말 삭제하시겠습니까? (연관된 패키지와 디바이스에 영향이 있을 수 있습니다)"
3. 의존성 검사: 연결된 디바이스가 있으면 경고
4. 확인 → 삭제 수행
5. 성공 토스트 및 리스트 갱신

### 7. 정렬 (추가 기능)
- 컬럼 헤더 클릭 → 해당 컬럼 기준으로 오름차순/내림차순 정렬
- 예: Title 클릭 → 알파벳순 정렬, Devices 클릭 → 디바이스 수 순 정렬

## 컴포넌트 및 검증

### 필터 패널
- 필터 조건은 URL 쿼리 파라미터로 관리 (북마크/공유 지원)
- 초기화 버튼 제공: 모든 필터를 기본값으로 리셋

### 체크박스 및 일괄 작업
- Select All 체크박스: 현재 페이지의 모든 항목 선택/해제
- 개별 체크박스: 각 항목 선택
- 선택된 항목 개수 표시 (예: "3 products selected")
- Bulk Actions는 최소 1개 이상 선택되어야 활성화

### 상태 표시
- 제품의 상태는 별도로 관리하지 않음

### 페이지네이션
- 기본 페이지 크기: 10개 (설정에서 변경 가능)
- 페이지 번호 클릭 시 해당 페이지로 이동
- Previous/Next 버튼으로 앞뒤 페이지 이동

## 에러 및 엣지 케이스 처리

### 제품 없음
- 필터 조건에 맞는 제품이 없을 경우:
  ```
  📭 No products found
  Try adjusting your filters or [Create New Product]
  ```

### 삭제 시 의존성 경고
- 제품에 연결된 디바이스가 있을 경우:
  ```
  ⚠️ Warning: This product has 45 active devices.
  Deleting it will affect all connected devices.
  [View Devices] [Cancel] [Delete Anyway]
  ```

### 일괄 작업 실패
- 일부 항목 실패 시:
  ```
  ✓ 3 products processed successfully
  ✗ 1 product failed (has active devices)
  [View Details]
  ```

## UX 노트
- 제품 리스트는 제품 관리의 중심 화면입니다
- 검색·필터링으로 원하는 제품을 빠르게 찾을 수 있어야 합니다
- 각 제품의 패키지 수와 디바이스 수를 한눈에 파악할 수 있습니다
- 일괄 작업(Bulk Actions)으로 여러 제품을 효율적으로 관리할 수 있습니다
- 각 제품에서 바로 편집/상세보기/삭제 작업으로 이동할 수 있어 작업 흐름이 원활합니다

## 반응형 고려사항
- 모바일: 테이블을 카드 뷰로 전환, 필터는 접을 수 있는 패널로 변경
- 태블릿: 사이드바를 토글 가능하도록 변경, 컬럼 간격 조정
- 데스크톱: 전체 테이블 뷰, 모든 기능 표시

## 접근성
- 키보드 내비게이션 지원: Tab, Enter, Space로 모든 기능 접근 가능
- 스크린리더 지원: 각 컬럼, 버튼, 상태에 적절한 레이블 제공
